# Audio Storytelling

Use this skill to shape podcast episodes for listening retention.

## Core Guidance

- Design for theater of the mind and audio intimacy.
- Match BPM to episode type and shift pace intentionally.
- Use open loops, cold opens, callbacks, cliffhangers, and intentional silence.
- Keep transitions clear and make each segment earn the listener's attention.

## Useful Patterns

- Classic three-act structure
- Mystery reveal structure
- Parallel comparison structure
- List frame structure
- Conversational exploration structure
