# Podcast Growth

Use this skill to package and promote podcast episodes.

## Core Guidance

- Optimize metadata for Apple Podcasts, Spotify, and YouTube Podcasts.
- Write promotional copy with a clear hook, insight, key moment, and CTA.
- Build a 7-day promotional calendar around release day.
- Use searchable keywords naturally.
- Keep the promotion message focused on why the listener should care.
