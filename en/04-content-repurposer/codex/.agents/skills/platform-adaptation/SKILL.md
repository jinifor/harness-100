# Platform Adaptation

Use this skill to rewrite a message for a platform's native form.

## Core Guidance

- Keep the core message identical while changing the format.
- Respect platform-specific length, structure, and algorithm preferences.
- Use the source element-to-platform mapping to choose what to emphasize.
- Avoid copy-pasting the same prose across every platform.
- Keep the CTA consistent even when the wording changes.
