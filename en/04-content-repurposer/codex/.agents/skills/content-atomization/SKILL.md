# Content Atomization

Use this skill to split source content into reusable units.

## Core Guidance

- Map the source into claims, data, stories, processes, quotes, questions, contrasts, and definitions.
- Score each atom by independence, emotional impact, shareability, and visualizability.
- Prioritize the atoms that can survive across multiple formats.
- Recombine the atoms differently for blog, social, and presentation outputs.
- Add missing context only when it improves conversion or clarity.
